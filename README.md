# 🛡️ VetoLayer: The Fiduciary Intercept

**VetoLayer** is a Just-In-Time (JIT) governance protocol that prevents "Financial Hallucinations" in autonomous agentic workflows. 

### 🚀 The Mission
As agents move from **Information** (Chat) to **Action** (Transactions), the risk of unauthorized capital movement grows exponentially. VetoLayer provides a hardened intercept between the Agent and the Wallet.

### 🏗️ Core Architecture
- **JIT Intercept:** Real-time request monitoring for agentic spend.
- **Biometric Veto:** Push-notification authorization for the human-in-the-loop.
- **RLS-Secured Ledger:** Every decision is written to an immutable audit trail, secured by Postgres Row Level Security.
